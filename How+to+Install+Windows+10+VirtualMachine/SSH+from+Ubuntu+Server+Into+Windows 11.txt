Here i will Teaching You how to SSH from Ubuntu Server into Windows 11 

First you have to check if sshd is installed on the Windows machine 

Open Powershell in Administartive mode 

type sshd

if you see some messages or codes that show that ssh is installed 

what next is to start the sshd 

On windows you have to make sure that your OpenSSH allow ssh both public ip and Private Ip 

in Powershell open with Administartive mode 

Type this : Get-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0        # show the state of the ssh 

Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0      # start the OpenSSH

Start-Service -Name sshd       # start the sshd 

Set-Service -Name sshd -StartupType Automatic # this will start the sshd upon starting your computer 

New-NetFirewallRule -Name sshd -DisplayName 'OpenSSH Server (sshd)' -Enabled True -Direction Inbound -Protocol TCP -Action Allow -LocalPort 22   # allow Inbound and outbund ip 

ssh your_windows_username@windows_ip 

Enter your password of your Windows Computer
